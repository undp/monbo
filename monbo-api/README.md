### Deploy on fly.io

1. Create an account on [fly.io](https://fly.io).
2. Install the fly.io CLI.
3. Log in to your account using:
   ```sh
   fly auth login
   ```
4. Each time you want to deploy to fly.io, use:
   ```sh
   fly deploy
   ```
